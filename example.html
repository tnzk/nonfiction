Nonfiction - the controllable chart library

The library, "Nonfiction", provides some charts that allows the user to change the values on the chart. The edited values will be sent as HTTP parameters same as other form-fields.

The charts available are listed below(with its how-to-use):

Slider

<script type="text/javascript">  slider( 'slider', 200, 40, 100, { north: '#009', south: '#500', border: '#005', borderhighlight: '#00f'}); </script>
<input type="button" value="Check the value!" onclick=" alert($('#nf_slider').val());" />
